    Проверка модулей на сервере

    В игре на сервере с E2fullExtensions — откроется панель со статусом каждого модуля.

    e2fullextensions_modules

    Консоль (~) или чат. Статусы: Активен · Зависимости есть, E2 не загружен · Не найдено.

    Как пользоваться

    1. На сервере введи e2fullextensions_modules — см. проверку модулей.
    2. Убедись, что нужные аддоны установлены в garrysmod/addons/.
    3. Найди функцию в разделе Справочник API или готовый пример ниже.
    4. Нажми Copy под блоком кода и вставь в E2-редактор.
    5. Подключи Wire-входы и выходы к своей схеме.
    6. Если функция «не найдена» — перезагрузи Lua (lua_refresh в консоли) или перезапусти карту.
    ID отряда: у каждого отряда есть числовой идентификатор. Игрок получает его через entity:getSquadID(); если игрок не в отряде — возвращается −1. Для проверок надёжнее использовать ID, чем имя: имя можно найти через getSquadIDByName или findSquadIDByPart, либо выбрать отряд по позиции в getAllSquadIDs().

    Модули и проверка зависимостей

    Все E2-расширения в одном аддоне. В консоли или чате: e2fullextensions_modules — откроется панель со статусом каждого модуля.

    Модуль E2fullExtensionsExtensionWorkshop-аддон / проверка

    Squad / EVENTsquad_e2plus, eventsystem_e2Simple Squad Menu, EVENT system

    Статусы в панели

    • Активен — модуль загружен, функции доступны в E2
    • Зависимости есть — аддон на месте, но E2 ещё не перезагружен
    • Не найдено — нужный аддон/API отсутствует

    Примеры — отряды

    Проверка: игрок в нужном отряде

    Отряд задаётся по индексу в списке getAllSquadIDs() или по имени через Wire-вход.

    @name Squad Check
@inputs SquadName:string SquadIndex Check
@outputs Allowed SquadID
@persist TargetSquadID
@trigger Check

TargetSquadID = -1
if(SquadIndex >= 1) {
    local N = 0
    foreach(_, ID:number=getAllSquadIDs()) {
        N += 1
        if(N == SquadIndex) { TargetSquadID = ID }
    }
}
if(TargetSquadID == -1) {
    TargetSquadID = getSquadIDByName(SquadName)
    if(TargetSquadID == -1) { TargetSquadID = findSquadIDByPart(SquadName) }
}
SquadID = TargetSquadID
if(Check) {
    Allowed = (doesSquadExist(TargetSquadID) & owner():isInSquad(TargetSquadID)) ? 1 : 0
}

    Список всех отрядов на экран

    Выводит название и число участников каждого отряда.

    @name Squad List
@outputs Screen:string
@trigger

local T = ""
foreach(_, ID:number=getAllSquadIDs()) {
    T = T + getSquadName(ID) + " (" + getSquadMemberCount(ID):toString() + ") | "
}
Screen = T

    Реакция на вход и выход из отряда

    Чип выполняется, когда кто-то вступает в отряд или покидает его.

    @name Squad Events
@outputs EventText:string
@trigger

if(first()) { runOnSquadEvent(1) }
if(squadEventClk()) {
    local P = squadEventPlayer()
    local Act = squadEventType() == 1 ? "joined" : "left"
    EventText = P:name() + " " + Act + " squad " + P:getSquadID():toString()
}

    Проверка участника активного ивента

    Wire-выход Allowed = 1, если ивент идёт и игрок в списке участников.

@inputs Check
@outputs Allowed

    Примеры — оплата

    Запрос оплаты по кнопке

    При нажатии Buy отправляет игроку окно оплаты; при успехе включает выход Paid.

    @name Shop E2
@inputs Buy
@outputs Sold
@persist Price
@trigger Buy

if(first()) { Price = 75; runOnPointPaySuccess(1) }
if(Buy) {
    local C = owner()
    if(C:getPointBalance() >= Price) { C:pointPayRequest(Price, "Purchase") }
}
if(pointPaySuccess() == 1) {
    Sold = 1
    timer("reset", 1000)
    runOnPointPaySuccess(0)
    runOnPointPaySuccess(1)
}
if(clk("reset")) { Sold = 0 }

    Запрос оплаты деньгами (Sandbox Money / DarkRP)

    @name Money Shop
@inputs Buy
@outputs Sold
@persist Price
@trigger Buy

if(first()) { Price = 500; runOnMoneyPaySuccess(1) }
if(Buy) {
    local C = owner()
    if(C:getMoneyBalance() >= Price) { C:moneyPayRequest(Price, "Purchase") }
}
if(moneyPaySuccess() == 1) {
    Sold = 1
    timer("reset", 1000)
    runOnMoneyPaySuccess(0)
    runOnMoneyPaySuccess(1)
}
if(clk("reset")) { Sold = 0 }

    Комбинации

    Условие: отряд или ивент, иначе оплата

    Если игрок в нужном отряде или участник ивента — выход Access. Иначе запрашивается оплата поинтами.

    @name Access Logic
@inputs Request SquadIndex
@outputs Access Paid
@persist SquadID Price
@trigger Request

if(first()) { Price = 100; runOnPointPaySuccess(1); SquadID = -1 }
if(SquadIndex >= 1) {
    local N = 0
    foreach(_, ID:number=getAllSquadIDs()) {
        N += 1
        if(N == SquadIndex) { SquadID = ID }
    }
}
if(Request) {
    local OkSquad = doesSquadExist(SquadID) & owner():isInSquad(SquadID)
    local OkEvent = isEventRunning() & owner():isEventParticipant()
    if(OkSquad | OkEvent) { Access = 1 }
    else { owner():pointPayRequest(Price, "Access fee") }
}
if(pointPaySuccess() == 1) {
    Paid = 1
    Access = 1
    runOnPointPaySuccess(0)
    runOnPointPaySuccess(1)
}
